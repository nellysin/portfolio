Django intro

The MVC paradigm seperates into 3 categories. (Model/View/Control)

Model- is a duplicated data.(Law) View- is the presentation of the data/result.(People) Control- is the logic of the system/ control the flow going between the model and view.(Police)

The web framwork is a collection of what you use. The library is to simplify common web development operations.

Useing framwork has a template so saves time than creating the app yourself.
